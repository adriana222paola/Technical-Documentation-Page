# Technical Documentation Page
# Pokédex 

## Contenido
- `index.html`: Página principal con la estructura de la Pokédex.
- `styles.css`: Archivo de estilos personalizados.
- Uso de **Bootstrap 5.3.3** para mejorar el diseño y la responsividad.

## Objetivo
El propósito de este proyecto es:
- Practicar **HTML** y **CSS**.
- Implementar una interfaz visual atractiva y funcional, con un diseño similiar a un Pokédex. 
- Preparar la base para futuras integraciones con **JavaScript** y **APIs externas** (ej. [PokeAPI](https://pokeapi.co/)).

## Características
- Tarjetas individuales para cada Pokémon con:
  - Nombre
  - Tipo
  - Estadísticas básicas (HP, ataque, defensa)
  - Habilidad especial
  - Evoluciones
- Diseño responsivo con Flexbox y media queries.
- Interactividad inicial con botones y eventos JavaScript.
- Buenas prácticas de accesibilidad (uso de `alt`, `nav`, y soporte para teclado).
